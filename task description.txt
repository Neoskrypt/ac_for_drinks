1. Создать консольную прогу для ведения файловой базы учёта напитков в ресторане.
Требования:
1.1. Запуск команды делается так:
$ python resto.py cmd args
Где cmd -  название команды, args -  произвольный список аргументов, которые могут быть у конкретной команды.
Никакого вечного цикла и менюшек интерактивности : запустил, обработал, закрыл. 
1.2. Список команд:
- list: выводит на экран список объектов заданного типа в бд. Значение аргумента: drinks (список напитков) , reserves (запасы напитков). 
- add: добавить в бд. Аргумент: drink (новый напиток), reserve <drink name> (пополнить запас напитка, в литрах). 

1.3. По каждому напитку храним: name, amount. 
1.4. Создать и использовать исключения для:
- напиток уже существует
- некоректное  значение объёма напитка для пополнения 
- напиток не найден 
- ошибка в переданных аргументах 
1.5. Данные по напиткам храним в бинарном файле. Использовать библиотеку shelve. Ключом выступает название напитка, значением - словарь с параметрами напитка.
